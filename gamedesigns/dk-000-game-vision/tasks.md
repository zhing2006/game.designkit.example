## 任务概述

本任务清单将 `000-game-vision` 的全局设计产物按用户故事组织，确保每个用户故事独立可审查。全局型规范关注游戏的宏观设计方向（"为什么"和"是什么"），产出4个全局设计文档。

**设计产物**：
1. game-vision.md - 游戏愿景文档
2. core-loop.md - 核心玩法循环设计
3. world-setting.md - 世界观设定文档
4. emotional-curve.md - 情感曲线设计

**用户故事优先级**：
- **P1（必需）**: US1（核心循环）、US2（视觉沉浸）、US4（新手引导）
- **P2（重要）**: US3（阵营系统）

- 两份文档的设计意图与 spec.md 用户故事1的验收场景一致
- 所有设计决策可追溯到 research.md 的研究结论

## Phase 6: 用户故事 4 - 新手到硬核玩家的渐进式引导（P1）

**故事目标**: 定义游戏的情感曲线设计（emotional-curve.md），确保"渐进式引导"和"难度分层"的设计意图清晰，支持"可接近的硬核体验"设计支柱。

**独立测试标准**:
- emotional-curve.md 明确定义七阶段单局情感流（准备期→探索期→搜刮高峰→战斗期→逃离期→撤离成功/失败）
- emotional-curve.md 清晰描述长期情感（积累满足感、技巧成长感、战略层满足）
- emotional-curve.md 详细定义氛围营造手段（视觉、听觉、叙事）
- 情感设计与 spec.md 用户故事4的验收场景一致（新手3-5局理解核心、10小时内首次满载）
- 情感设计可追溯到 research.md 第4节（情感曲线与氛围营造研究）和第6节（可接近性与硬核平衡研究）

### 设计文档编写

- [X] T014 [US4] 编写 emotional-curve.md - 情感曲线设计 /mnt/g/AI/XinDong/xd_spec_design/test/docs/game-vision/emotional-curve.md
  - 内容来源: spec.md 第46-56行（情感曲线部分）+ research.md 第4节（情感曲线与氛围营造研究）+ research.md 第6节（可接近性与硬核平衡研究）
  - 必需章节:
    - **单局情感流（七阶段）**:
      1. 准备期（低紧张 2/10）: 2-5分钟，安全区整理装备、规划路线
      2. 探索期（中紧张 4/10）: 3-8分钟，进入危险区域，警惕但可控
      3. 搜刮高峰（高兴奋 7/10）: 1-3分钟（多次），找到稀有物资的"屯屯鼠"快感
      4. 战斗期（极高紧张 9/10）: 1-5分钟，遭遇敌人，生死一线
      5. 逃离期（焦虑与希望 8/10）: 2-5分钟，携带物资冲向撤离点
      6. 撤离成功（强释放 10/10）: 30秒-2分钟，成就感爆棚
      7. 撤离失败（挫败但不绝望 3/10）: 1-3分钟，保留技能经验，渴望复仇
    - **长期情感（跨局）**: 积累满足感（仓库从空到满）、技巧成长感（从新手到老手）、战略层满足（阵营势力扩大）
    - **氛围营造手段**:
      - 视觉氛围: Lumen动态光影（夕阳穿过破窗、手电筒在隧道中）、Nanite高细节废墟（锈蚀、涂鸦、植物侵蚀）
      - 听觉氛围: 环境音效（风声、雨声、僵尸低吼）、音乐设计（探索时的低沉弦乐、战斗时的激烈鼓点）
      - 叙事氛围: 环境叙事（翻倒的婴儿车、褪色的广告牌、遗弃的避难所）讲述无声的故事
    - **情感强度曲线图**: 横轴时间（0-30分钟）、纵轴情感强度（0-10）
    - **为每个阶段定义关键设计元素**: 音效、音乐、光照、事件触发
    - **渐进式引导设计**:
      - 新手教程: 在"安全的危险区域"（僵尸少、撤离点近）引导玩家完成第一次"搜-打-撤"
      - 难度分层: 低/中/高难度区域，撤离成功率70-80% / 50-60% / 30-40%
      - 失败容错机制: 保底装备、借贷任务、技能永不丢失
      - 可选辅助: 挫败模式检测、推荐撤离路线、僵尸感知范围可视化
  - 设计支柱对齐: 体现支柱4（可接近的硬核体验）
